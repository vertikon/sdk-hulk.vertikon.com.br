# Exemplos SDK-HULK

⚠️ **Nota:** Estes exemplos são apenas para referência e documentação. Eles não fazem parte do build do módulo principal.

Os exemplos demonstram como usar o SDK-HULK para criar módulos, mas não devem ser compilados diretamente. Eles são incluídos apenas como documentação de uso.

## Estrutura

- `inventory_module/` - Exemplo completo de módulo de estoque
- `simple_module/` - Exemplo mínimo de implementação

## Como usar

Estes exemplos devem ser adaptados e movidos para o projeto principal (`hulk-core`) onde serão compilados junto com a implementação do Core.
